# NextWave - Landing Page

Projeto desenvolvido como teste prático.


## 🚀 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript
* PHP
* MySQL

## 📋 Funcionalidades

* Landing page responsiva
* Animações ao rolar a página
* Menu mobile
* Formulário funcional com envio via AJAX
* Integração com banco de dados MySQL

## 💾 Banco de dados

O arquivo `banco.sql` está incluído no projeto.

Para utilizar:

1. Abra o phpMyAdmin
2. Crie um banco chamado `nextwave`
3. Importe o arquivo `banco.sql`

## ▶️ Como rodar o projeto

1. Instale o XAMPP
2. Coloque a pasta em:

   ```
   xampp/htdocs/
   ```
3. Inicie Apache e MySQL
4. Acesse:

   ```
   http://localhost/nextwave-site
   ```
